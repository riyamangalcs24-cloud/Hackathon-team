# LearnHub – Modern Online Learning Platform

LearnHub is a modern, responsive e-learning platform built with **React** and **Vite**. It provides an engaging user experience with dynamic course listings, pricing plans, testimonials, authentication popups, and a fully validated contact form.

---

## 📖 Overview

LearnHub is designed to provide students with an intuitive platform to explore courses, learn about instructors, compare subscription plans, and contact the organization with ease. The application follows a modular component-based architecture and is fully responsive across desktop, tablet, and mobile devices.

# ⚙️ Installation

## Clone the repository

```bash
git clone https://github.com/riyamangalcs24-cloud/Hackathon-team.git
```

## Navigate to the project

```bash
cd Hackathon-team/learnhub
```

## Install dependencies

```bash
npm install
```

## Run the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
